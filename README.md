# progress
 1. 로봇 멀티모달 상호작용 파이프라인

-본 로봇은 사용자의 음성과 표정을 인식하고 자연스럽게 대화할 수 있도록 구현한 파이프라인 코드를 담고 있습니다. 크게 두 가지 핵심 로직으로 구성되어 있습니다.

# 주요 파이프라인

1. STT - LLM (Exaone) - TTS 파이프라인
- 사용자의 음성을 텍스트로 변환(STT)한 뒤, 엑사원(Exaone)모델을 거쳐 상황에 맞는 답변을 생성하고 로봇의 음성(TTS)으로 출력하는 기본 음성 대화 시스템입니다.

2. Mediapipe - LLM - TTS 파이프라인
- 카메라로 사용자의 얼굴 랜드마크와 표정을 실시간으로 읽어내고(Mediapipe), 그 감정 상태를 LLM이 판단하여 적절한 위로나 공감이 담긴 대사를 음성(TTS)으로 출력하는 시각 기반 시스템입니다.
